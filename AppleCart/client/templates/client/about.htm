ABOUT US



Developed as the part of Web Technologies Mini Project, "AppleCart" transforms the way in which shopping is done. Using "AppleCart", one can buy all the grocery items that he needs via online, making the shopping secure, convenient and efficient in terms of Pricing. This project is implemented using django for back end and HTML, CSS and BootStrap for front end. Using AppleCart one need not go to shops to buy grocery items as the items are delivered to your doorsteps. Hence AppleCart is India's very own Online Grocery Store
